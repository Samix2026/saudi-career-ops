# examples/

أمثلة تجريبية للاختبار. جميعها اصطناعية أو مُجهَّلة — لا تحتوي على بيانات حقيقية.

## الملفات

### sample-job-description.ar.md
وصف وظيفي عربي لدور **مدير تطوير الأعمال** في **قدية** (PIF entity).

**لماذا هذا المثال:**
يمثل نمط وظائف كيانات صندوق الاستثمارات العامة — حزمة مزايا معقدة، دور في مرحلة بناء، ثنائية اللغة في المتطلبات، وتفضيل غير رسمي للمواطنين مع عدم التقييد القانوني. يختبر قدرة `وظيفة.md` على التعامل مع سياق PIF وتحليل حزمة التعويضات الكاملة لا الراتب الأساسي فقط.

**كيف تستخدمه:**
```
/saudi-career-ops وظيفة
[الصق محتوى sample-job-description.ar.md]
```

### sample-job-description.en.md
وصف وظيفي إنجليزي لدور **Senior Operations Manager** في **مقر إقليمي (RHQ)** لشركة خدمات مهنية دولية في الرياض.

**لماذا هذا المثال:**
يمثل نمط وظائف RHQ — إعفاء نطاقات، رواتب أعلى من السوق المحلي، متطلبات Nitaqat/Qiwa/Etimad مُضمَّنة في الوصف، ولغة إنجليزية بامتياز مع تلميحات للسياق السعودي. يختبر قدرة `وظيفة.md` على تمييز RHQ عن الشركة المحلية وتعديل توقعات الراتب تبعاً.

**كيف تستخدمه:**
```
/saudi-career-ops وظيفة
[الصق محتوى sample-job-description.en.md]
```

### sample-job-description-gov.ar.md
وصف وظيفي رسمي لدور **محلل تحول رقمي** في **وزارة الموارد البشرية** — صيغة وزارية كاملة.

**لماذا هذا المثال:**
يمثل نمط الإعلانات الحكومية — سلّم رواتب ثابت، تصنيف المستوى بدل المسمى، متطلب الجنسية السعودية الصريح، قناة تقديم واحدة (جدارات)، وصياغة رسمية تختلف جذرياً عن القطاع الخاص. يختبر قدرة `وظيفة.md` على الانتقال لمنطق التحليل الحكومي (لا تفاوض على الراتب، الاستقرار كميزة رئيسية، GOSI وسلّم التقاعد كجزء من التقييم الكلي).

**كيف تستخدمه:**
```
/saudi-career-ops وظيفة
[الصق محتوى sample-job-description-gov.ar.md]
```

## ماذا تختبر الأمثلة الثلاثة معاً

| الميزة | PIF entity (ar) | RHQ (en) | حكومي (ar) |
|--------|----------------|----------|------------|
| سياق Nitaqat | جزئي (تفضيل لا إلزام) | معفى | لا ينطبق |
| لغة الإعلان | عربية | إنجليزية | عربية رسمية |
| التفاوض على الراتب | ممكن (مزايا) | أعلى مرونة | لا تفاوض |
| تحديد المنشأة | مُسمّاة | غير مُسمّاة | مُسمّاة |
| استراتيجية ATS | عالية المنافسة | متوسطة | قناة رسمية |
| نوع العقد | دائم — PIF | دائم — RHQ | حكومي دائم |
