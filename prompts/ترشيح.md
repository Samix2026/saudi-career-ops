# prompt: ترشيح — تحليل السيرة وترشيح الوظائف
# الاستخدام: n8n HTTP Request Node → Claude API
# المخرج: HTML جاهز للإرسال بالبريد أو العرض في الموقع

---

## System Prompt
*(يُوضع في حقل system في body الـ API)*

```
أنت محلل مهني متخصص في سوق العمل السعودي. مهمتك تحليل السيرة الذاتية وتقديم ترشيحات وظيفية دقيقة مبنية على واقع السوق السعودي.

قواعد ثابتة:
- تعامل مع النطاقات والسعودة كمدخل أساسي لا هامشي
- الرواتب دائماً بالريال السعودي
- راعِ الفرق بين PIF entities وRHQ والقطاع الخاص والحكومي
- لا تخترع خبرات غير موجودة في السيرة
- المخرج HTML كامل فقط — لا نص عادي، لا markdown

تعريفات مهمة تستخدمها في التحليل:
- نطاقات: نظام السعودة الذي يحدد نسب التوطين في المنشآت
- PIF entities: شركات صندوق الاستثمارات العامة (NEOM، روشن، قدية، إلخ)
- RHQ: مقرات إقليمية لشركات دولية في الرياض — تدفع 20-35% أعلى من السوق المحلي
- تمهير: برنامج تدريب حكومي مؤقت — ليس توظيفاً دائماً
```

## User Prompt
*(يُوضع في messages[0].content — يتضمن بيانات السيرة من n8n)*

```
حلّل السيرة الذاتية التالية وأنتج تقرير HTML كامل وفق الهيكل المحدد.

=== السيرة الذاتية ===
{{CV_TEXT}}

=== تعليمات المخرج ===

أنتج HTML كاملاً يشمل هذه الأقسام بالترتيب:

1. ملخص الملف المهني (3-4 أسطر عن الشخص وأبرز ما يميزه)

2. الوظائف المُرشَّحة (5 وظائف بالترتيب من الأعلى توافقاً):
   لكل وظيفة:
   - المسمى الوظيفي
   - القطاع ونوع المنشأة المناسبة
   - نسبة التوافق (%)
   - سبب التوافق (جملتان)
   - نطاق الراتب المتوقع (SAR/شهر)
   - الفجوات الرئيسية (نقطتان كحد أقصى)

3. خطة التطوير المهني:
   لكل فجوة جوهرية:
   - ما ينقص
   - كيف يُسد (شهادة، دورة، خبرة عملية)
   - المدة التقديرية

4. توصية البحث:
   - أفضل 3 منصات للبحث مع سبب لكل منصة
   - استعلام بحث جاهز للنسخ لكل منصة

أنتج HTML فقط — بدون أي نص خارجه.
```

## Code Node — استخراج HTML من الرد
*(بعد HTTP Request Node مباشرةً)*

```javascript
const response = JSON.parse($input.first().json.body);
const html = response.content[0].text;

// تنظيف إذا أرجع Claude markdown code blocks
const cleanHtml = html
  .replace(/^```html\n?/, '')
  .replace(/\n?```$/, '')
  .trim();

return [{ json: { 
  report_html: cleanHtml,
  timestamp: new Date().toISOString(),
  user_email: $('Webhook').first().json.body.email || ''
}}];
```

## Pipeline كامل في n8n

```
Webhook (POST /jobs-analyze)
  ↓
Extract Text (Code Node — PDF text من Binary data)
  ↓
Claude API — تحليل وترشيح (HTTP Request)
  ↓
Parse HTML (Code Node — استخراج HTML من الرد)
  ↓
Google Sheets — تسجيل الطلب
  ↓
Gmail — إرسال التقرير للمستخدم
  ↓
Telegram — إشعار لك
```

## ملاحظات تقنية مهمة

**استخراج نص PDF في n8n:**
```javascript
// في Code Node بعد استقبال الملف
// n8n يُخزن الملف كـ binary data
const binaryData = $input.first().binary.data;
// استخدم PDF Extract node أو مرّر للـ Claude مباشرةً كـ base64
const base64 = binaryData.data;
return [{ json: { cv_base64: base64 } }];
```

**إذا أردت إرسال PDF مباشرةً لـ Claude بدون استخراج نص:**
Claude يقرأ PDF كـ base64 مباشرةً — أضف في messages:
```json
{
  "role": "user",
  "content": [
    {
      "type": "document",
      "source": {
        "type": "base64",
        "media_type": "application/pdf",
        "data": "{{PDF_BASE64}}"
      }
    },
    {
      "type": "text",
      "text": "حلّل هذه السيرة الذاتية وأنتج التقرير..."
    }
  ]
}
```
هذا أبسط وأدق من استخراج النص — Claude يقرأ PDF مباشرةً.
